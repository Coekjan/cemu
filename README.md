# CEMU

A simple full system emulator.

## ISAs Support

- RISC-V
  - RV64IMA
  - Support M-Mode, S-Mode and U-Mode
  - Sv39 MMU and TLB
  - RISC-V CLINT
  - RISC-V PLIC
  - Capable of booting S-Mode SMP Linux with OpenSBI

## Devices Support

- Xilinx UARTLite
